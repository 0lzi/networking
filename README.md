# Networking Labs

This repo is for a playground for all my Networking shenanigans.

## Labs

- [vxlan](./vxlan) - This lab is a basic vxlan setup without evpn using [vyos](vyos.io) as the L3 devices and the PC/Client.
- [vyos-lab](https://github.com/0lzi/vyos-lab) - Some basic [Ansible](https://docs.ansible.com/projects/ansible/latest/index.html) to configure Vyos in my homelab
- [FS.com Switches automation](https://github.com/0lzi/fs-switch-automation) - Some Ansible to configure/template [FS.com]() switches, utilising `Jinja` Templating.
